# Database Models

## User
- Account details, hashed password, preferences, wishlist, appointments, orders.

## Product
- Multi-language description, collection, occasion tags, images/videos, featured status.

## Appointment
- Scheduling, location details, collection preference, status lifecycle, WhatsApp confirmation flag.

## Order
- Itemized products, amount fields, payment method/status, order status, shipping details.

## Testimonial
- Client quote, rating, featured visibility.
